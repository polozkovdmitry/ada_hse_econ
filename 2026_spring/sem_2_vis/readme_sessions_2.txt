Документация датасета sessions_2.csv

витрина про сессии в приложении

ключи
- user_id - уникальный пятизначный идентификатор пользователя
- session_start_dttm - время начала сессии

атрибуты
- desktop_flag - 1 для ноутбука/пк, 0 для телефона
- satisfaction_metric - метрика удовлетворённости, <0 не удовлетворён, >0 удовлетворён, принимает любые значения на прямой
-- задаётся как нормальное (2, 1) для desktop_flag=1, и нормальное (1, 1) для desktop_flag=0
- session_duration_min - длительность сессии (в минутах)
- purchase_flag - флаг совершения покупки за данную сессию
- purchase_amount_rub - сумма покупки за данную сессию (в рублях)
