Документация датасета sessions_2

таблица про сессии в приложении

хранится помесячно

Внимание! не содержит поюзерные характеристики, для этого надо джойн с users_2.scv по ключу user_id

ключи
- user_id - уникальный пятизначный идентификатор пользователя
- session_start_dttm - время начала сессии

атрибуты
- desktop_flag - 1 для ноутбука/пк, 0 для телефона
- satisfaction_metric - метрика удовлетворённости, <0 не удовлетворён, >0 удовлетворён, принимает любые значения на прямой
-- задаётся как нормальное (2, 2) для desktop_flag=0, и нормальное (1, 2) для desktop_flag=1
